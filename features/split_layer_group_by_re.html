正则拆分图层组¶

使用 正则表达式 包含图层名称进行匹配， 并对每一个规则匹配的图层创建 Shuffle 和 Merge 节点。

支持同时为多个节点进行分割，不同节点中匹配同样规则的图层 Shuffle 将被作为同一个 Merge 的输入。

在菜单 编辑 - 正则分割图层组 执行命令对选中节点进行此操作。

点击后会弹出配置对话框。

正则表达式规则每行一条。 默认规则为匹配不区大小写的 _col 和 _sdw 结尾的图层。

填写匹配规则后点 OK 按钮执行

对话框会自动读取上一次的设置，存储在 ~/.nuke/com.wlf-studio.storage-48017c27.db 中。
